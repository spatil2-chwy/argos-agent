# Argos Voice Interaction

Argos voice interaction is direct microphone audio in, direct speaker audio out,
same-session transcription for observability/memory, and realtime tool calling
inside one persistent OpenAI Realtime session.

Use the focused docs instead of repeating the runtime details here:

- `launch.md` for setup and bring-up.
- `realtime_turn_flow.md` for audio admission, turn boundaries, playback, and interruption.
- `prompting_and_history.md` for what the model sees.
- `observability.md` for latency and cost logs.
- `speaker_recognition.md` for speaker ownership and voice enrollment.
- `interaction_display.md` for Puffle screen state and subtitles.
